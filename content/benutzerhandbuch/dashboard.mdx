# Dashboard

Das Dashboard zeigt eine Übersicht der wichtigsten Kennzahlen:

- **Mitarbeiter** - Gesamtanzahl der Mitarbeiter im System
- **Teams** - Anzahl der Teams
- **Aktive Buchungen** - Aktuell laufende Buchungen (Status: Bestätigt)
- **Anstehend** - Zukünftige bestätigte Buchungen

Das Dashboard ist die Startseite nach dem Login und bietet einen schnellen Überblick über den aktuellen Stand der Ressourcenplanung.
